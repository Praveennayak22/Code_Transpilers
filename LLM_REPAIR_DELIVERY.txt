                              ↓
┌─────────────────────────────────────────────────────────────────────────┐

│                                                                          │
│ ① Check if code compiles                                               │
│    ✅ Success? Done!                                                   │
│    ❌ Failed? Send to LLM...                                           │

│ ② Extract compiler error                                               │
│    "undefined reference to 'printf'"                                    │
│                                                                          │
│ ③ Send to DeepSeek-v3.2 with context                                  │
│    Code + Error + Source language → HTTP call                           │
│                                                                          │
│ ④ LLM fixes code                                                        │
│    DeepSeek analyzes problem → returns fixed code                       │
│                                                                          │
│ ⑤ Retry compilation                                                     │
│    ✅ Success? Done! Use fixed code                                    │
│    ❌ Failed? Repeat (max 3 times)                                     │
│                                                                          │
│ Output: fixed_code + repair_attempts + llm_tokens_used                  │
